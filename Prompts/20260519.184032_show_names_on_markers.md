# Implementation Plan - Dynamic Named Map Markers

We will enhance the existing `MainActivity.java` by upgrading the programmatic marker drawing method `createTeardropMarkerBitmap`. 

## 1. Key Objectives
- Draw beautiful, high-contrast, premium-styled floating text badges (capsules) with custom names ("Red", "Green", "Blue") directly above each map marker pin.
- Ensure the label text is perfectly centered and highly legible.
- Connect the floating name badge to the teardrop marker head using a clean vertical stem matching the pin's color.
- Make the label and marker fully dynamic, moving seamlessly together across the map as the marker positions update.

## 2. Technical Strategy
MapLibre's marker API allows setting a custom icon via a programmatic `Bitmap`. Since the markers move inside a `Handler` loop using `Marker.setPosition(LatLng)`, the entire custom bitmap moves automatically with the marker. By embedding the name badge directly into the marker's `Bitmap` canvas, we guarantee perfect synchronization without lagging or alignment issues.

### Canvas Grid Refactoring (`160 x 192` Pixels)
To accommodate the name badges floating above the markers:
- **Canvas Size**: `160` (width) x `192` (height) pixels.
- **Marker Pin Tip**: Bottom-centered at `(80, 160)` to allow clean anchoring.
- **Marker Pin Circle**: Centered at `(80, 96)` with a radius of `34` pixels.
- **Connecting Stem**: Drawn from the capsule bottom `(80, 48)` to the pin head top `(80, 62)`.
- **Text Capsule**: Drawn from `y = 12` to `y = 48` (height of `36` pixels, rounded corner radius of `18` pixels) and dynamically sized based on text length to handle arbitrary names.

### High-End Visual Design System (Dark-Mode Capsule Accent)
- **Background**: Solid modern charcoal color (`#1E1E24`).
- **Border**: Thick accent border (`3px`) matching the exact color of the pin (Crimson Red, Vibrant Green, or Sky Blue).
- **Text**: Premium bold white text, vertically and horizontally centered.
- **Shadows**: Soft drop shadows under the pin tip and the capsule badge to create a 3D overlay depth.

## 3. Code Modifications
We will refactor `createTeardropMarkerBitmap(int color)` to `createTeardropMarkerBitmap(int color, String text)` and pass the corresponding labels (`"Red"`, `"Green"`, `"Blue"`) during marker initialization in `onCreate`.

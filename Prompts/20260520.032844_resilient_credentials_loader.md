# Robust Resilient Credentials JSON Loader

## Accomplishments
Improved loading robust-ness and error tolerance for the Matrix credentials JSON file located at `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/matrix_credentails.json`.

## Technical Optimizations

1. **Independent Fail-Safes**:
   - Wrapped each separate JSON key parsing action (`matrix_homeserver`, `matrix_username`, `matrix_password`, `matrix_token`, `matrix_display_name`, `matrix_polling_period`) in isolated `try-catch` blocks.
   - If any single field is missing or contains an unexpected/unusual format in your JSON file, it will no longer abort the entire load process. All other successfully parsed keys will continue to load and populate correctly.

2. **Polymorphic Polling Period Handling**:
   - Handled both strictly numeric integer structures and string representations for `"matrix_polling_period"` dynamically, parsing them cleanly under any config variations without causing exceptions.

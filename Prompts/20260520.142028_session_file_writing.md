# Matrix Session Message File Writing

## Accomplishments
Successfully implemented file output handling inside `MainActivity.dispatchMatrixMessage()` to store incoming Matrix location coordinates directly on the local filesystem rather than displaying them inside floating toast messages.

## Technical Details

1. **Clean Message Splitting & Parsing**:
   - Replaced legacy toast notification popup logic.
   - Built a split-delimiter parser targeting incoming message strings. Uses double-escaped literal Pipe character split pattern (`body.split("\\|")`) to extract:
     - `Latitude`
     - `Longitude`
     - `<someColor>` (custom color parameter or cobalt default `#1976D2`)

2. **Session Storage File Handlers**:
   - Created robust location handlers supporting both `/sdcard/Vypeensoft/Friends_Location_Tracker/sessions/` directory directly and fallback parameters matching emulator external mounts.
   - Saves formatted coordinates with an active overwrite flag set (`new java.io.FileWriter(file, false)`) to cleanly update location properties:
     - Format: `cleanSender|Latitude|Longitude|<someColor>`

   - Compiling successfully: **BUILD SUCCESSFUL**!

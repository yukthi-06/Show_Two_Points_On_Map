# Clear Sessions on Room Change

## Accomplishments
Successfully implemented automated cleanup of tracking session logs inside the sessions folders whenever the active room is switched to a different choice under the "Groups/Rooms" section.

## Technical Details

1. **Active Room Switch Detection**:
   - Enhanced `onRoomSelected()` inside `GroupsRoomsActivity.java` to detect if the newly selected room index is different from the currently active room choice.
   - Triggers `clearSessionsFolder()` when a different room is activated.

2. **Complete Dual Directory Purge**:
   - The deletion routine `clearSessionsFolder()` scans and deletes all files inside BOTH potential target sessions folders to gracefully support any spelling variations:
     - `/sdcard/Vypeensoft/Friends_Location_Tracker/sessions/`
     - `/sdcard/Vypeensoft/Freinds_Location_Tracker/sessions/`
   - Handles environment fallback paths using `Environment.getExternalStorageDirectory()` cleanly.

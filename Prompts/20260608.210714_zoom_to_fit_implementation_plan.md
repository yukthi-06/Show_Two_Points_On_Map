# Zoom to Fit Participant Markers Plan

We will add a floating ImageButton on the map view that displays a `[   ]` (brackets/focus bounds) icon. When clicked, it will compute the bounding box of all active participant markers and zoom/animate the camera to fit them with standard padding.

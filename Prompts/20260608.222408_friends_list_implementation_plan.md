# Add Friends Checklist Activity Plan

We will add a new menu option in the navigation drawer named "Friends", below "Matrix Rooms". Clicking it opens `FriendsActivity` containing a two-column checkbox list of friends. Clicking "Track Them...." saves the selected friends in SharedPreferences, and `MainActivity` will filter the map markers so that only the selected friends are shown.

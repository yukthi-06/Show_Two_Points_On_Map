### 3. Friends Activity Screen
- Created [FriendsActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/java/com/vypeensoft/friendtracker/FriendsActivity.java) which dynamically loads friends from `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/friends.json`.
- Supports JSON lists defined as Arrays `["Shibu", "Syama", ...]` or Objects with a `"friends"` array property.
- Falls back to the 9 hardcoded default names (Shibu, Syama, Yukthi, Lakshya, Suseela, Binu, Sheeba, Neha, Nisha) if the JSON file is missing or empty.
- Automatically calculates column division and programmatically populates the columns with CheckBoxes.
- Loads existing selections from SharedPreferences (`tracked_friends` key) on startup.
- Saves the set of checked friends to SharedPreferences upon clicking `"Track Them...."` and finishes the activity.

### 4. Manifest Registration
- Registered the activity in [AndroidManifest.xml](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/AndroidManifest.xml) with `MainActivity` declared as its parent activity.

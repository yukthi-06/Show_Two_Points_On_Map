# Cloudflare Settings Page Plan

We will add a new menu item in the navigation drawer named "Cloudflare" directly below "Matrix Rooms". Clicking it opens `CloudflareSettingsActivity` containing settings to configure `cloudflareUrl` and `cloudflarePollingInterval`. Clicking "Save" persists these values to `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/cloudflare.json` in JSON format.

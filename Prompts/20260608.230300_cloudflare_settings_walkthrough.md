# Walkthrough - Cloudflare Settings Screen

We have implemented a Cloudflare Settings screen where users can configure their backend server URL and polling interval, persisting the configurations into a local JSON settings file.

- Added a "Cloudflare" menu option directly below "Matrix Rooms" in [activity_main.xml](file:///x:/Projects_X:0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/res/layout/activity_main.xml).
- Bound and configured click listener for this menu option in [MainActivity.java](file:///x:/Projects_X:0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/java/com/vypeensoft/friendtracker/MainActivity.java) to start `CloudflareSettingsActivity`.

### 2. Cloudflare Settings Layout
- Created [activity_cloudflare_settings.xml](file:///x:/Projects_X:0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/res/layout/activity_cloudflare_settings.xml) containing:
  - An AppBar / Toolbar.
  - A scrollable area containing two text fields: Server URL (`edit_cloudflare_url`) and Polling Interval (`edit_cloudflare_polling_interval`).
  - A Save button (`btn_save_cloudflare`) styled premiumly at the bottom.

### 3. Cloudflare Settings Screen Code
- Created [CloudflareSettingsActivity.java](file:///x:/Projects_X:0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/java/com/vypeensoft/friendtracker/CloudflareSettingsActivity.java):

  - On Save, parses input, validates values, and writes a JSON object with keys `"cloudflareUrl"` and `"cloudflarePollingInterval"` to `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/cloudflare.json`.

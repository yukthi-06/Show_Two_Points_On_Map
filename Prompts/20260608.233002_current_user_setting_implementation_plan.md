# Add Current User Setting Plan

We will add a new text field for "Current User" in the Settings screen (`MapSettingsActivity`). The value will be loaded from and saved to SharedPreferences, and exported/imported to `/sdcard/Vypeensoft/Friends_Location_Tracker/settings/map_settings.json` via `SettingsPersistenceManager`.

- Add a new `TextInputLayout` containing a `TextInputEditText` for the "Current User" under a new section header "User Settings".

### Map Settings Activity

#### [MODIFY] [MapSettingsActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/java/com/vypeensoft/friendtracker/MapSettingsActivity.java)

- Bind the view in `onCreate()`.
- Load the value in `loadConfig()` and save it in `saveConfig()`.

### Settings Persistence Manager

#### [MODIFY] [SettingsPersistenceManager.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/java/com/vypeensoft/friendtracker/util/SettingsPersistenceManager.java)

# Cloudflare Tracking Flow Logging Implementation Plan

We will add comprehensive logging to the Cloudflare tracking flow inside `LocationService.java`. We will also customize the existing `AppLogger.java` to write logs to `/sdcard/Vypeensoft/Friends_Location_Tracker/logs/log_${yyyyMMdd.hhmmss}.txt` with fallback to `Environment.getExternalStorageDirectory()` if `/sdcard` directory writing is not available.

## User Review Required

> [!NOTE]
> The logging filename format requested is `log_${yyyyMMdd.hhmmss}.txt`. In Java's `SimpleDateFormat`, `hh` represents the 12-hour clock (1-12) and `HH` represents the 24-hour clock (00-23). We will use `HHmmss` for 24-hour formatting to ensure chronological file sorting and avoid filename collision/overwrite between AM and PM app launches, formatting it as `log_yyyyMMdd.HHmmss.txt`.

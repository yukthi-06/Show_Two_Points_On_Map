# Stop Tracking Button Integration Plan

We will add a prominent, high-visibility action button at the bottom of the main map screen that allows the user to start or stop background location tracking. When stopped, the app will completely stop sending JSON updates to the server.

## User Review Required

> [!NOTE]
> We will implement a toggle button overlaying the bottom center of the map.
> - When tracking is active, the button is **Red (Crimson)** with the text **"Stop Sending"** (or similar, customized for premium aesthetics).
> - When tracking is inactive, the button is **Green (Emerald)** with the text **"Start Sending"**.
> - This state is persisted in `SharedPreferences` so that background service starts/stops are respected across app relaunches.

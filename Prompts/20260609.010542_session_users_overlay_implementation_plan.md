# Session Usernames Overlay Implementation Plan

We will add a text overlay at the top of the map view in the main screen showing space-separated usernames of the current tracking session.

## User Review Required

> [!NOTE]
> The overlay text will appear at the top-center of the map with a translucent dark background and white text. It will automatically update in real-time as the session updates location files.

- Add a `TextView` named `tv_session_users` inside the `FrameLayout` overlaying the top part of the map.

# Friends Checkbox Overlay Implementation Plan

We will update the overlay text view (`tv_session_users`) at the top of the map to display the exact set of usernames currently selected in the friends checklist (stored in SharedPreferences), rather than only those whose location files are present.

## User Review Required

> [!NOTE]
> The overlay text will show all checked friends. If no friends are checked, the overlay will be hidden.

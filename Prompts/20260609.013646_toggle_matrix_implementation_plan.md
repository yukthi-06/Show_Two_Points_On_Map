# Enable/Disable Matrix Activity Implementation Plan

We will add a toggle option to the Matrix settings screen (Matrix Credentials Activity) to enable/disable all Matrix activity. When disabled, the app will not run any Matrix polling loop, and the background location tracking (GPS tracking) service will not run.

## User Review Required

> [!NOTE]
> A switch labeled **"Enable Matrix Activity"** will be added to the Matrix Credentials screen.
> - If checked (default), Matrix polling and background GPS tracking will function normally.
> - If unchecked, Matrix polling loops in `MainActivity` will be stopped, and the background `LocationService` (responsible for GPS tracking) will be stopped and prevented from running.

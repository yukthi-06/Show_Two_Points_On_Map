# Cloudflare Tracking Compatibility Implementation Plan

We will update the Matrix disabled mode logic to only disable Matrix-specific loops and communications (such as message polling and location posting to the Matrix homeserver) while allowing background GPS tracking and Cloudflare tracking loops to continue running unimpeded.

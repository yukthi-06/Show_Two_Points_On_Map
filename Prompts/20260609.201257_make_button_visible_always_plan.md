# [Make Button Visible Always]

The Start/Stop tracking button is not visible. This is because it is set to `gone` by default in the layout XML, and the visibility logic is only evaluated when permissions are granted and the movement loop runs. We need to set its default visibility to `visible` in the layout file and ensure it is kept visible in `MainActivity`.

## Proposed Changes

### Layout

#### [MODIFY] [activity_main.xml](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/res/layout/activity_main.xml)
- Change `android:visibility="gone"` to `android:visibility="visible"` (or remove the visibility attribute so it defaults to visible) for the `btn_toggle_tracking` Button.

### MainActivity

#### [MODIFY] [MainActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/01_GPS/Friends_Location_Tracker_APK_GITHUB_LATEST/app/src/main/java/com/vypeensoft/friendtracker/MainActivity.java)
- In `onCreate`, set `btnToggleTracking.setVisibility(android.view.View.VISIBLE)` right after finding the view.

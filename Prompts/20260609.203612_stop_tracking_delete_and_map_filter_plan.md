# User Prompt
when user click on Stop Tracking button, invoke end point /delete with sessionid and userName.  It returns the remaining users as response.
Show only current user on map and not  other users

# Goal Description
Modify the Android application to:
1. Invoke the `/delete` endpoint when the "Stop Tracking" button is clicked. This will clean up the current user's session from the server.
2. Process the returned list of remaining users, write their updated locations, and delete any stale session files.
3. Modify the map display logic to only show the current user on the map, filtering out all other users.

## User Review Required
No major user review required. Standard network call and UI behavior.
